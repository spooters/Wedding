# Marcus & Hayley's Wedding Planner

Edit this list any time. Each heading (##) becomes a section, and each line
starting with "- [ ]" becomes a task. Mark something done by changing
[ ] to [x] — or just tap it in the app once it's loaded.

## 12+ Months Before
- [x] Set the budget
- [x] Draft the guest list
- [ ] Book the venue
- [ ] Choose the wedding party
- [ ] Book photographer & videographer

## 6-9 Months Before
- [ ] Book caterer
- [ ] Order the dress & suit
- [ ] Book band or DJ
- [ ] Book florist
- [ ] Send save-the-dates

## 3-4 Months Before
- [ ] Send invitations
- [ ] Arrange transport
- [ ] Book hair & make-up trial
- [ ] Order wedding rings
- [ ] Plan the honeymoon

## 1 Month Before
- [ ] Confirm final numbers with venue & caterer
- [ ] Write vows and speeches
- [ ] Final dress/suit fitting
- [ ] Confirm all suppliers

## Week Of
- [ ] Pack for the honeymoon
- [ ] Give rings to best man/maid of honour
- [ ] Delegate a point-of-contact for the day
- [ ] Rehearsal & rehearsal dinner
